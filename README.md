# Jasmine Uyen Le — GRC Portfolio

This repository publishes the portfolio through GitHub Pages.

## Edit your content

1. Open `_data/portfolio.yml` on GitHub.
2. Select the pencil icon (**Edit this file**).
3. Change the text after the relevant label. Preserve the spacing at the beginning of each line.
4. Select **Commit changes**.

GitHub Pages rebuilds the website automatically after each commit. Updates usually appear within a few minutes.

Common updates:

- `hero` — headline, summary, and focus areas
- `proofs` — measurable outcomes
- `expertise` — services and capabilities
- `journey` — education, roles, projects, and AI experience
- `credentials` — certifications
- `knowledge` — podcast, speaking, and writing
- `closing` — contact invitation

## Edit the appearance

The visual theme is in `assets/css/style.css`. The main colors are at the top of that file under `:root`.

## Replace the social image

Upload a new 1200 × 630 image at `assets/og.png` using the same filename.

## Important

YAML uses indentation to organize content. Keep the existing indentation and use spaces rather than tabs.
